# torii-web

A [Torii](https://github.com/haratosan/torii) extension that fetches the content of a URL and returns the body text.

## Usage

Accepts a `url` parameter and returns the fetched page content as text.

## Configuration

| Variable       | Description   |
|----------------|---------------|
| `HTTP_PROXY`   | HTTP proxy    |
| `HTTPS_PROXY`  | HTTPS proxy   |
